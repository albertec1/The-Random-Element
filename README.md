# The Random Element

## Description

Fun 2D platformer for development subject at CITM
 
## Controls

 - w, a, s, d to move arround

 - space to jump.

 - K for the special ability

 - Debug keys: 	
  - F5: Save Game
  - F6: Load Game
  - F9: Show colliders, paths and walkable area
  - F10: GodMode
  - F11: 30FPS cap
